# Memory Folder

This folder is used to persist context across AI-assisted development sessions.

## Files

| File | Purpose |
|---|---|
| `SESSION_LOG.md` | Chronological log of what was done in each session |
| `PROJECT_OVERVIEW.md` | High-level summary of the project, modules, and features |
| `ARCHITECTURE.md` | Technical decisions, file locations, and infrastructure details |
| `NEXT_STEPS.md` | Planned work, potential improvements, and known issues |

## How to use
- **Start of session:** Read all files in this folder to understand prior context
- **During session:** Update `SESSION_LOG.md` with actions taken
- **End of session:** Update `NEXT_STEPS.md` with any new planned work or discovered issues
- **When architecture changes:** Update `ARCHITECTURE.md` accordingly
